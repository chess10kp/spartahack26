All notable changes to the "forcegraph" extension will be documented in this file.

## [0.0.1] - 2026-01-31

### Added
- AI-generated code challenges using OpenAI GPT-4
- Navigation challenges: Find functions, classes, and code patterns
- Modification challenges: Make targeted code changes
- Gamification system with points, streaks, and levels
- Rich interactive sidebar UI with real-time stats
- Celebration animations on challenge completion
- Hint system (2 hints per challenge)
- Auto-advance between challenges
- Smart workspace analysis that ignores test files and build artifacts
- Real-time verification of challenge completion
- Challenge history tracking
